# 1. What is TypeScript?

TypeScript is a superset of JavaScript that adds static typing.
💬 Human Explanation

👉 JavaScript has no type checking → errors come at runtime
👉 TypeScript catches errors during development

🎯 Interview Line

👉 “TypeScript helps write safer and more maintainable code by adding types on top of JavaScript.”

In real projects:
- You catch errors before running code
- Less bugs in production

let age: number = 25; // correct ✅
let age: number = "25"; // error ❌
